## Demo

Clone the repository and move into:

```console
$ git clone https://github.com/onursimsek94/todo-list.git
$ cd todo-list
```

Install dependencies:

```console
$ npm install
```

Run the server:

```console
$ npm start
or
$ npm run start:chrome
```

Open your browser and go to [http://localhost:3001/](http://localhost:3001/)
